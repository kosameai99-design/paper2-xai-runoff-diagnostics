# Historical and deterministic IG modes

The manuscript results were generated by the historical tester, which temporarily placed the model in training mode during Integrated Gradients evaluation. With `output_dropout = 0.4`, this activates dropout while Captum evaluates the integration path.

The release audit compared this workflow with an eval-mode implementation that disables cuDNN only for recurrent backward operations:

- 3 basins and 3 dates per basin (9 samples).
- Historical repeatability median Spearman: 0.97619.
- Historical versus deterministic pooled Spearman: 0.97619.
- Pooled maximum grouped-share difference: 0.00465.
- Maximum sample-level grouped-share difference: 0.01735.
- Deterministic repeat maximum difference: 0.
- Median absolute Captum convergence delta: 0.05422 historical; 0.000107 deterministic.

The audit passed the predefined rank and share thresholds. Existing manuscript results were retained. The released IG runner defaults to deterministic mode and offers `--mode historical` for compatibility.
